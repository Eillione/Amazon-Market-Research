# Elione Amazon Market Research Skill

专业的亚马逊市场调研工具，提供完整的市场分析报告，包括关键词调研、细分市场、产品属性分析、竞品分析、用户画像、产品定义和产品策略等功能。

## 功能特性

### 📊 12大调研模块

| 模块编号 | 模块名称 | 功能描述 |
| :--- | :--- | :--- |
| 1 | US基础数据表 | 美国市场基础数据汇总 |
| 2 | 产品属性表 | 产品核心属性信息收集 |
| 3 | 产品属性分析 | 产品属性深度分析 |
| 4 | 品牌分析 | 品牌市场定位与竞争力 |
| 5 | 美国市场数据调查表 | 美国市场专项调研 |
| 6 | 市场趋势 | 市场热度与季节性分析 |
| 7 | 竞品分析 | 竞争对手深度剖析 |
| 8 | 用户画像表 | 目标用户群体分析 |
| 9 | Review分析表 | 用户评价深度挖掘 |
| 10 | Q&A分析表 | 常见问题汇总分析 |
| 11 | QA数据源 | Q&A数据来源管理 |
| 12 | 供应链调查表 | 供应链信息调研 |

### ✨ 核心亮点

- **品牌分析强制交互**：先提供核心竞争品牌列表，用户选择后深入分析品牌体量、核心交易单品、淡旺季等
- **VOC用户声音分析**：调用VOC工具多渠道收集用户反馈，分析基本需求、反向需求、期望需求等
- **海关数据供应链分析**：结合核心竞品海关数据，分析供应商来源和成本结构
- **数据真实性保障**：不编造硬数据，不确定数据标注"待验证"或"假设"
- **决策导向输出**：输出直接落到产品经理可执行决策

## 快速开始

### 触发方式

```text
请帮我调研 [产品关键词] 市场
调研 [产品关键词] 亚马逊市场
分析 [产品关键词] 市场
```

### 输入参数

**必选参数：**
- `调研关键词`: 目标产品的核心关键词（英文）
- `目标市场`: 默认美国市场，可选择其他市场

**可选参数：**
- `时间范围`: 数据采集的时间跨度（默认近1年）
- `竞品数量`: 分析的竞品数量（默认10-20个）
- `数据深度`: 分析深度级别（基础/标准/深度）
- `输出格式`: 报告输出格式（表格/图表/综合报告）

### 使用示例

```text
请帮我调研 maternity pillow 市场
```

```text
调研无线扫描仪亚马逊市场
竞品数量: 15
数据深度: 深度
```

## 调研流程

1. **关键词市场总览** → 产品属性收集与分析 → 品牌分析
2. **品牌分析强制交互**：系统提供核心竞争品牌列表，用户选择后深入分析品牌体量、核心交易单品、淡旺季等
3. **美国市场专项调研** → 市场趋势分析 → 竞品分析
4. **用户画像构建** → Review深度挖掘 → Q&A分析
5. **供应链调研**（含海关数据分析）→ 输出完整报告


## 数据与分析原则

### 市场口径与分类原则
- 市场口径优先使用Amazon关键词自然搜索结果
- 细分市场分类基于关键词搜索结果前几页自然位产品集合
- 竞品ASIN必须嵌入Amazon商品页超链接

### 数据真实性原则
- 不编造硬数据，不确定数据标注"待验证"或"假设"
- 所有数据来源必须可追溯
- 关键数据采用多种工具交叉验证

### 决策导向原则
输出必须落到产品经理可执行决策：
- 进入哪个细分市场
- 为什么选择该细分市场
- 做什么SKU
- 用什么卖点
- 如何定价
- 成本是否成立
- 怎么上线验证

## 输出格式

- **默认输出**: PDF文件
- **文件命名**: `市场调研报告_[关键词]_[日期].pdf`
- **保存位置**: 当前工作目录下的output文件夹
- **内容包含**: 完整调研分析报告、图表、数据表格

## 支持工具

- **关键词工具**: MerchantWords、卖家精灵、sorftime、sif
- **趋势分析**: Google Trends、Keepa、JS、KeywordtoolDominator
- **竞品分析**: 卖家精灵、BSR榜单、Google搜索
- **知识产权**: USPTO、EUIPO、行业报告
- **供应链**: 海关数据、品牌官网、企业信息查询平台

## 注意事项

1. 调研数据来源于公开渠道，仅供参考
2. 建议结合多种工具交叉验证数据准确性
3. 市场数据具有时效性，建议定期更新调研结果
4. 复杂调研可能需要较长时间，请耐心等待

## 技术限制

- 单次调研最多分析20个竞品
- 历史数据最多查询近5年
- 部分数据依赖第三方工具接口
